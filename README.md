# Tarta Fit — Web

Versão web (single-file) do app de treino & dieta **Tarta Fit**.

Treino com tracking de séries/RPE/PRs, dieta com macros e busca Open Food Facts,
jejum intermitente, corpo (IMC e % gordura), evolução com gráficos, atividade
por GPS (Leaflet) e Coach de IA (Claude). Dados no banco local do navegador
(IndexedDB), com exportar/importar.

🔗 **App:** https://rodolfot.github.io/tarta-fit-web/

> Recursos nativos (Health Connect, GPS em segundo plano, push, sync na nuvem)
> só existem no app Android.
